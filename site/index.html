Running in 60 seconds

Download and open

Run the Windows installer or unzip the portable build. No Python needed for Desktop.

02

Press Run Demo

FORGE reads demo_input.md, creates action_items.md, and streams each execution step live.

03

Give a real task

Try: "Analyze this folder and write a report" or "Create a Python script and run it."
