# FORGE — Humanitarian Agent System Architecture
## نظام تشغيل الوكيل الذكي الإنساني — المعمارية التقنية الكاملة
### إصدار 1.0 | مطوّر لمشروعات TREN Studio

---

## 1. الرؤية الإنسانية والهدف السامي (Humanitarian Vision & Objective)

تأسس مشروع **FORGE (Free Open Reasoning & Generation Engine)** برؤية واضحة ومستقلة: **إتاحة الذكاء الاصطناعي الوكيل (Agentic AI) المتقدم مجاناً وبدون قيود لجميع البشر، ومساعدة المطورين والمبدعين في كافة أنحاء العالم على زيادة إنتاجيتهم وتأمين لقمة عيشهم دون الحاجة لدفع اشتراكات مكلفة للشركات الكبرى.**

يقوم النظام على المبادئ التالية:
- **مجاني بالكامل (100% Free)**: لا جدران دفع (No paywalls)، ولا إعلانات، ولا اشتراكات خفية.
- **سيادة البيانات والخصوصية (Data Sovereignty & Privacy)**: قاعدة المعرفة المحلية والذاكرة تعمل بالكامل على جهاز المستخدم عبر SQLite.
- **التنوع والمرونة (Provider Neutrality)**: دمج أكثر من 20 نموذجاً مجانياً ومفتوح المصدر لضمان عدم الاحتكار والاستمرارية.

## 2. البنية المعمارية للنظام (System Architecture Layers)

يتكون نظام تشغيل FORGE من 5 طبقات رئيسية تعمل بتناسق تام لإنتاج ردود ذكية، وتنفيذ عمليات حقيقية على نظام التشغيل:

```
                  ┌──────────────────────────────┐
                  │      User Interface / CLI    │
                  └──────────────┬───────────────┘
                                 │
                                 ▼
                  ┌──────────────────────────────┐
                  │    Identity Guard Layer      │  ◄── (Branding Anchor & Sanitizer)
                  └──────────────┬───────────────┘
                                 │
                                 ▼
                  ┌──────────────────────────────┐
                  │  Smart Router & Quota Guard  │  ◄── (20+ Free Providers / Speed Routing)
                  └──────────────┬───────────────┘
                                 │
                                 ▼
                  ┌──────────────────────────────┐
                  │   Dynamic Agent Council      │  ◄── (AgentFactory: 20+ Roles / DNA)
                  └──────────────┬───────────────┘
                                 │
                                 ▼
                  ┌──────────────────────────────┐
                  │  Skill Registry & Executors  │  ◄── (File, Shell, Browser, APIs)
                  └──────────────────────────────┘
```

### أ. طبقة حماية الهوية (Identity Guard Layer)
- **مُعترض الهوية الفوري (Instant Response)**: يعترض الأسئلة المباشرة المتعلقة بالهوية والتحيات ويرد عليها في أقل من 10ms دون إرسالها إلى الـ APIs لتوفير الحصة (Quota).
- **مُطهر الردود (Response Sanitizer)**: يفحص الردود القادمة من النماذج الخارجية ويستبدل أي ادعاء بالتبعية لشركات كبرى (مثل Google أو OpenAI) ويؤكد هوية FORGE التابعة لـ **TREN Studio**.

### ب. طبقة التوجيه الذكي وحارس الحصص (Smart Router & Quota Guardian)
- **التوجيه حسب السرعة (Speed-based Routing)**: تصنيف المهام إلى (سريعة، عادية، معقدة) وتعيين مهلة استجابة تصاعدية (Progressive Timeout).
- **حارس الحصة (Quota Guardian)**: يتتبع عدد الكلمات (Tokens) والطلبات المستخدمة لكل مفتاح API ويقوم بالتحويل التلقائي للمزود البديل فور اقتراب الحصة من النفاد.

### ج. مجلس الوكلاء الديناميكي (Dynamic Agent Council)
- **مصنع الوكلاء (AgentFactory)**: يحلل خطوات الخطة ويقوم بإنشاء وكلاء متخصصين ديناميكياً (مثل `PythonSpecialist`, `DBConsultant`, `WebScraper`) ويصل عددهم إلى أكثر من 20 دوراً.
- **الحمض النووي للمحادثة (Conversation DNA)**: سياق مشترك متنقل يحفظ أهداف الجلسة، والقرارات المتخذة، وتفضيلات المستخدم، ويتم تمريره لكل وكيل جديد عند تبديل النموذج لضمان استمرارية التفكير.

### د. سجل المهارات والمنافذ الخارجية (Skill Registry & Tool Registry)
- **المهارات المحلية (Local Skills)**: تنفيذ التعليمات البرمجية، تعديل الملفات بأمان، والتحكم في المتصفح المحلي (Browser Execution).
- **سجل الأدوات الخارجية (External Tool Registry)**: تكامل آمن عبر OAuth لإدارة الملفات والاتصالات (Google Workspace, Slack, Notion, GitHub).

## 3. تكامل الـ 20+ نموذج مجاني (Multi-Model Integration)

يستمد FORGE قوته من التنوع. يدعم النظام دمج وتشغيل النماذج المجانية عبر المزودين التاليين:

1. **Groq Cloud**: تشغيل نماذج LLaMA 3.3 بسرعات فائقة.
2. **Google AI Studio**: واجهة Gemini المجانية مع حدود استدعاء كبيرة (1M tokens/day).
3. **DeepSeek API**: نماذج الاستدلال الفائق (Reasoner/R1).
4. **OpenRouter (Free Tier)**: بوابة تجمع نماذج Qwen و LLaMA و Mistral المجانية.
5. **Together AI**: استضافة النماذج المفتوحة وتوفير بدائل تشغيل سريعة.
6. **NVIDIA NIM**: نماذج الحوسبة السحابية المتطورة للتعليمات البرمجية المعقدة.
7. **Cloudflare Workers AI**: تشغيل النماذج على الحافة (Edge) بشكل خفيف ومستقر.
8. **Mistral AI**: نماذج Codestral والمهام الإبداعية.
9. **Anthropic (BYOK)**: واجهة خطوط الدفاع الأمامية (Claude).
10. **OpenAI (BYOK)**: نماذج GPT-4o و GPT-mini.
11. **Ollama (Local Path)**: تشغيل نماذج محلية 100% على جهاز المستخدم دون الحاجة للإنترنت.

## 4. ميثاق الأمان والاستقلالية (Safety & Autonomy Contract)

- **التحقق من الآثار الجانبية (Side-Effect Validation)**: لا يتم تشغيل أي أمر ذي تأثير خارجي (مثل إرسال بريد إلكتروني، أو تعديل ملف حساس، أو كتابة كود على النظام) دون موافقة المستخدم الصريحة (Human-in-the-loop).
- **التراجع التلقائي (Automatic Rollback)**: في حال فشل خطوة في الخطة، يقوم النظام بالتراجع عن التعديلات البرمجية لضمان سلامة بيئة عمل المستخدم.
